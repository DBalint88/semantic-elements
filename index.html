15 HTML Elements (6/15)

Semantic elements in HTML consist of any element whose tag describes its function. This is in contrast to non-semantic elements such as div. The point is that they not only tell the browser how to render the content, but also describe to anyone reading the code what it is they're looking at. (7/15)
